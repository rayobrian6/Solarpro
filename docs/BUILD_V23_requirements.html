🔴 P1 — Critical (Incorrect / Code Violation Risk)

P1

Battery connected directly to MSP bus — WRONG for Enphase

Current BUILD v22 draws a dashed blue line from battery directly to the MSP bus. For Enphase AC-coupled systems, the battery must connect to the IQ System Controller 3 (BUI), which then connects to the MSP via a dedicated backfeed breaker.

sld-professional-renderer.ts NEC 705.12(B) misrepresented
P1

Battery/generator/ATS data NOT sent from page.tsx to SLD API

batteryBackfeedA, generatorBrand/Kw, atsBrand/AmpRating, backupInterfaceId are never included in the SLD POST body. Components cannot appear regardless of what the user configures.

app/engineering/page.tsx Components never render
P1

Enphase IQ System Controller 3 (BUI/ATS) not rendered on SLD

The IQ SC3 is the central component of every Enphase battery system. It is service-entrance rated, functions as an ATS, and is the connection point for the battery, backup panel, and optional generator. It must appear on the right side of the MSP.

sld-professional-renderer.ts Missing required component
P1

Generator + generator ATS not appearing on SLD

Even when generator and ATS are configured in the UI, they do not appear on the SLD because the data is never sent from page.tsx to the API. The generator ATS must appear between the utility meter and MSP (service entrance position).

page.tsx · route.ts Missing required component

🟠 P2 — High (Missing Required Components)

P2

Enphase backup sub-panel not shown when hasBackupPanel=true

The backup sub-panel (critical loads panel) must appear connected to the IQ SC3 load port, not to the ATS output. The current implementation connects it to the generic ATS output which is incorrect for Enphase.

sld-professional-renderer.ts
P2

backupInterfaceId not tracked in config or sent to SLD API

The BackupInterface entity (IQ SC3, Tesla Gateway, Generac PWRmanager) is in the equipment database but is never selected in the UI config, never stored in state, and never sent to the SLD API. This is the key missing link for manufacturer-specific BUI rendering.

page.tsx · equipment-db.ts
P2

120% rule calculation does not include battery backfeed amps

NEC 705.12(B) requires that the sum of all backfeed breakers (PV + battery) does not exceed 20% of the main panel rating. The current calculation only includes the PV backfeed breaker.

sld-professional-renderer.ts · page.tsx
P2

Electrical sizing page missing battery/generator/ATS data points

Battery backfeed breaker size, generator output amps, generator OCPD, ATS amp rating, and NEC 702.5 compliance status are not shown on the electrical sizing page.

app/engineering/page.tsx

2.1 Enphase Battery System — Correct Topology

⚠ CRITICAL ERROR IN BUILD v22: The battery is drawn with a dashed line directly to the MSP bus. This is incorrect for ALL Enphase AC-coupled systems. The battery must connect to the IQ System Controller 3 (BUI), which is the transfer switch and gateway device.

Current (WRONG)

MSP Bus ←──── Battery (dashed blue line directly to MSP bus)

Required (CORRECT — Enphase AC-Coupled)

Utility Meter → MSP → [Backfeed Breaker] → IQ System Controller 3 (BUI/ATS)
                                                        │
                                          ┌─────────────┴──────────────┐
                                          │                            │
                                    BATTERY port                  LOAD port
                                          │                            │
                                 IQ Battery 5P/10T/3T         Backup Sub-Panel
                                 (LFP, 240V AC, IP55)         (Critical Loads)

Updated 120% Rule (NEC 705.12(B)) — Includes Battery

Total Backfeed = PV Backfeed Breaker + Battery Backfeed Breaker
120% Rule:     Main Panel Rating × 1.2 ≥ Main Panel Rating + Total Backfeed
Simplified:    Total Backfeed ≤ Main Panel Rating × 0.2

Example: 200A panel, 30A PV breaker, 20A battery breaker
  Total backfeed = 30 + 20 = 50A
  200 × 0.2 = 40A  →  50A > 40A  →  ⚠ FAIL — panel upgrade required

2.2 Generator + Generator ATS — Correct Topology

Note: The generator ATS is a SEPARATE device from the Enphase IQ SC3 BUI. When both battery and generator are configured, the Enphase IQ SC3 handles battery backup switching while a separate generator ATS (Generac RXSW100A3, Kohler RXT-100, etc.) handles generator transfer at the service entrance.

Required Placement

[Utility Service] → [Generator ATS] → [MSP Main Breaker]
[Generator] ──────→ [Generator ATS]

Position: Generator ATS between utility meter and MSP (service entrance)
Generator: Below and left of the ATS
